# Q 弹按键与果冻卡片效果使用说明

文件：

```text
jelly_ui_effects.gd
```

适用：Godot 4.x。

## 1. 挂载方式

把 `jelly_ui_effects.gd` 放进新项目，例如：

```text
res://scripts/ui/jelly_ui_effects.gd
```

然后在需要使用 UI 动画的场景中创建一个节点：

```gdscript
var jelly := JellyUiEffects.new()
add_child(jelly)
```

也可以把它做成 Autoload。

## 2. 普通按键 Q 弹

适合音乐键、返回键、普通菜单按钮。

```gdscript
jelly.attach_button_jelly(my_button)
```

效果：

- 按下：横向鼓起、纵向压扁；
- 松开：纵向回弹；
- 最后回到原始比例。

## 3. Yes / No 类按钮

适合 18+ 页面、确认弹窗按钮。

```gdscript
func _on_yes_pressed() -> void:
    jelly.play_choice_button_pulse(yes_button, false)

func _on_yes_confirmed() -> void:
    jelly.play_choice_button_pulse(yes_button, true)
```

`confirmed=false` 是第一次点击的选中效果。  
`confirmed=true` 是第二次确认时更强一点的效果。

## 4. Start Game 这种变大按钮

适合按钮第一次点击后变大、第二次点击才进入。

```gdscript
var selected := false

func _on_start_pressed() -> void:
    var target_size := Vector2(320, 60)
    var target_position := Vector2(60, 720)

    if not selected:
        selected = true
        jelly.play_grow_button_pulse(start_button, target_size, target_position, false)
        return

    jelly.play_grow_button_pulse(start_button, target_size, target_position, true)
    start_game()
```

如果切换主题或退出这个状态，需要恢复：

```gdscript
jelly.reset_grow_button(start_button, Vector2(200, 60), Vector2(120, 720), true)
selected = false
```

## 5. 选关卡卡片轮播

假设你有一组 `Control` 卡片：

```gdscript
var cards: Array = [card_01, card_02, card_03, card_04]
var selected_index := 0
var direction := 1
```

初始化：

```gdscript
jelly.layout_jelly_cards(cards, selected_index, false)
```

切换到下一张：

```gdscript
direction = 1
selected_index = posmod(selected_index + 1, cards.size())
jelly.layout_jelly_cards(cards, selected_index, true, direction)
```

切换到上一张：

```gdscript
direction = -1
selected_index = posmod(selected_index - 1, cards.size())
jelly.layout_jelly_cards(cards, selected_index, true, direction)
```

拖拽时的果冻预览：

```gdscript
jelly.update_jelly_card_drag(cards, selected_index, drag_pixels)
```

松手后再根据拖拽距离决定是否切换，然后调用 `layout_jelly_cards()`。

## 6. 可以调的参数

在 `jelly_ui_effects.gd` 顶部可以改：

```gdscript
const BUTTON_JELLY_SQUASH := Vector2(1.13, 0.84)
const BUTTON_JELLY_STRETCH := Vector2(0.96, 1.13)
const CARD_SWIPE_DRAG_MAX_OFFSET := 118.0
```

想更 Q 弹：把横向数值加大、纵向数值压低。  
想更稳：把数值往 `Vector2.ONE` 靠近。
